# Image Prompt Templates

Copy-paste ready prompts organized by use case. Customize the bracketed sections.

---

## Photorealistic — Natural

```
A [subject] in [location], [time of day]. Shot on a [lens mm] lens with
[aperture]. [Lighting description]. The mood is [mood]. Background shows
[background details]. No text overlays.
```

**Example:**
```
A woman hiking through a misty old-growth forest trail in the Pacific Northwest,
early morning. Shot on a 35mm lens with f/2.8 shallow depth of field. Soft
diffused light filtering through the canopy creates god rays. The mood is
peaceful and contemplative. Background shows towering Douglas fir trees fading
into fog. No text overlays or watermarks.
```

---

## Photorealistic — Studio

```
Professional studio photograph of [product/subject] on a [background].
[Lighting setup]. [Camera angle]. Clean, [adjective] composition with
[specific detail]. Shot for [context: e-commerce, editorial, catalog].
```

**Example:**
```
Professional studio photograph of a pair of white leather sneakers on a
seamless light gray background. Three-point lighting with key light at
45 degrees, soft fill, and subtle rim light. Slightly elevated camera angle.
Clean, minimal composition with a soft shadow underneath. Shot for an
e-commerce product page.
```

## Product Mockup

```
[Product] mockup shown in [context]. [Device/packaging] is [position/angle].
Screen displays [content description]. Background is [setting]. Realistic
shadows and reflections. [Additional detail].
```

**Example:**
```
MacBook Pro mockup on a clean wooden desk in a bright home office. The laptop
is open at a 45-degree angle facing slightly left. Screen displays a dashboard
UI with charts and metrics. Background shows a blurred bookshelf and a small
plant. Realistic shadows and reflections on the desk surface.
```

## Editorial Illustration

```
[Art style] illustration of [concept/scene]. [Color palette]. [Composition].
[Mood/feeling]. Used as a [context: blog header, magazine, book cover].
```

**Example:**
```
Flat vector illustration of a team collaborating around a large table with
sticky notes and laptops. Warm pastel color palette: peach, sage green, soft
blue, cream. Bird's eye view composition. Feeling of creative energy and
teamwork. Used as a blog article header image about product management.
```

## Social Media Post

```
Eye-catching social media graphic for [platform]. [Visual description].
Bold [color] background. Text overlay reads "[exact text]" in [font style]
at [position]. [Aspect ratio note]. Designed to stop the scroll.
```

**Example:**
```
Eye-catching Instagram post graphic. A split-screen showing a cluttered desk
on the left transforming into an organized, minimal workspace on the right,
connected by a glowing arrow. Bold gradient background from deep purple to
electric blue. Text overlay reads "SIMPLIFY" in bold white sans-serif centered
at the bottom. 1:1 square format. Designed to stop the scroll.
```

## Hero Image

```
Cinematic wide-angle hero image for [page type]. [Scene description].
[Lighting]. [Color mood]. Aspect ratio 16:9. Space for text overlay on
[left/right/center]. [Style reference].
```

**Example:**
```
Cinematic wide-angle hero image for a SaaS landing page. An abstract
representation of data flowing through a glowing neural network — streams
of light in blue and purple against a dark gradient background. Soft
ambient glow lighting. Cool-toned with electric accent highlights. Aspect
ratio 16:9. Open negative space on the left side for headline text overlay.
Modern, premium tech aesthetic.
```

## Thumbnail

```
YouTube/article thumbnail, bold and high-contrast. [Subject] in [setting].
[Expression/action]. Large text reads "[TEXT]" in [color] with [effect:
outline, shadow, glow]. Background is [simple/blurred/gradient].
Designed for maximum clickability at small sizes.
```

**Example:**
```
YouTube thumbnail, bold and high-contrast. A person looking surprised at
a laptop screen with their mouth open. Dramatic side lighting in warm orange.
Large text reads "10X FASTER" in bright yellow with a thick black outline.
Background is a blurred dark office. Red arrow pointing at the screen.
Designed for maximum clickability at small sizes.
```

## 3D Render

```
3D render of [subject] with [material/texture]. [Lighting setup].
[Camera angle]. [Background/environment]. Rendered in [style: Blender,
Cinema 4D style, etc.]. [Color palette].
```

**Example:**
```
3D render of abstract floating geometric shapes — spheres, torus, and cubes
with glossy metallic and matte clay materials. Soft studio lighting with
colored accent lights: warm orange from the left, cool blue from the right.
Slightly elevated camera angle. Clean gradient background from light gray to
white. Rendered in a modern Cinema 4D style. Pastel accent colors on select
surfaces.
```

## Pattern / Texture

```
Seamless tileable pattern of [elements]. [Style: hand-drawn, geometric,
organic]. [Color palette]. [Scale: small/medium/large repeat]. Clean edges
for tiling. Background color: [color].
```

**Example:**
```
Seamless tileable pattern of small tropical leaves and flowers — monstera,
palm fronds, hibiscus. Hand-drawn watercolor style with visible brush
texture. Color palette: forest green, sage, coral pink, gold accents.
Medium repeat scale. Clean edges for perfect tiling. Background color:
warm cream. No perspective distortion.
```

## Abstract Art

```
Abstract [style] artwork. [Technique: brushstrokes, splatters, gradients,
geometric]. [Color palette]. [Mood/energy: calm, chaotic, rhythmic].
[Composition: centered, asymmetric, flowing]. [Texture: smooth, rough,
layered]. No recognizable objects.
```

**Example:**
```
Abstract expressionist artwork. Bold sweeping brushstrokes and paint
splatters layered on textured canvas. Color palette: midnight blue, burnt
sienna, white, gold leaf accents. Mood is powerful and dynamic. Asymmetric
composition with tension between dense lower-left and open upper-right.
Thick impasto texture visible. No recognizable objects or text.
```

## Character Design

```
Character design sheet for [character description]. [Art style]. Show:
[front view / 3/4 view / expressions / poses]. [Color palette]. [Clothing/
accessories]. Clean background for easy extraction. [Personality note].
```

**Example:**
```
Character design sheet for a friendly robot barista — round body, expressive
LED eyes, small articulated arms holding a coffee cup. Pixar-style 3D art.
Show front view and 3/4 view side by side. Color palette: warm gray body,
teal accents, orange LED eyes. Wearing a tiny brown apron. Clean white
background. Personality: cheerful, helpful, slightly clumsy.
```
